# cron_jobs
List of config files that can be used in cron or Google Cloud Run jobs.
To add, remove, or edit Cloud Run jobs, please update the `_config/cloud_run_config.json` file with the job configuration and run the Python file `_config/update_cloud_run.py`.

## run-manual
If you want to run a config manually, you can add the configuration file here `debug/config_manual.json` and run the `run-manual` job on Cloud Run.
